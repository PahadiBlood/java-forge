# java-forge

> A personal collection of Java projects built to understand concepts deeply through hands-on practice.
> Each project is self-contained, focused, and tested.

&nbsp;

## About

Learning by building. Every project in this repository targets a specific set of Java concepts and grows in complexity over time, from Core Java fundamentals all the way through to Spring Boot, databases, and beyond.

**Stack:** Java 21 &nbsp;·&nbsp; JUnit 5 &nbsp;·&nbsp; Maven
